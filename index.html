BSc., Computer Engineering

Aug. 2016 - Jul. 2020

Graduation Project: "Detecting Abnormal Traffic Patterns in an IoT Tracking System," which designs and implements a low-cost IoT tracking system using an embedded device to collect the data to be processed on a cloud server to detect abnormal routes. Generally, the proposed system has three stages: (1) a light-weight and low-cost microcontroller that frequently collects its GPS data in terms of longitude and latitude and sends its status reports, (2) the cloud server will study and learn its traffic patterns and regular routes from these reports, (3) and a proposed approach to detect any non-recurrent traffic anomaly and let the server applies this approach to send automated alarm notifications to the user in the cases of traffic anomaly.
